# Writing your own methods

## sample method

```
def sayMoo
  puts 'mooooooo...'
end
sayMoo
```

```
mooooooo...
```

## method parameters

```
def sayMoo numberOfMoos
  puts 'mooooooo...'*numberOfMoos
end

sayMoo 3
```
```
mooooooo...mooooooo...mooooooo...
```
just like passing in a parameter when calling a function in JavaScript.
